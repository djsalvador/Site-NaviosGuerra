Neste exemplo de website: 
- adaptável a dispositivos móveis utilizando Bootstrap 
- é responsivo (mobile-first) 
- adaptável para diversos tamanhos de tela, dispositivos, SO e navegadores 
- os formulários são estilizados e contêm os inputs types corretos 
- maioria das tags de alguma classe bootstrap (img, table, li, etc) 
- utilizado carrousel, glyphicon e menus responsivos. 

OBS.: O projeto completo com a pasta de imagens encontra-se compactado.
